**AuraLens** is a plugin for BetterDiscord that adds a button to the user's context menu to view detailed information about their activity.
---

After activating the plugin, a new menu item appears when you right-click on a user: **â€œAuraLensâ€**.

**Features include:**

### Basic information
- ID and username   
- Discord account creation date (calculated based on snowflake ID)
- Number of messages and time spent in voice channels
- Last visited voice channel, server, and date 

### History avatars
- Gallery of uploaded user avatars    
- Convenient preview of images in full-screen mode 

### History Streams
- List of saved screenshots of user streams

### Live Stream Preview
- Display of the user's active stream (if they are currently broadcasting)   
- Automatically get a preview via `ApplicationStreamingStore`  

### History of servers
- Chronological list of recent logins and logouts from servers   
- Server icons, names, dates, and event type (Join/Leave)    

### Nikcname history

- Displaying old user nicknames with the servers where they were used 
- Indication of the time of the last change
